AObmen

AObmen is an automation software for currency exchange opertions.

More about AObmen software

I've created program for internal use in "Demetra Plus" company and it was successfully used in more than 40 kiosks during 2 years. Also we were trying to get certificate of National Bank of Ukraine to sell this software to other exchange companies and banks. AObmen created using Borland Pascal 7.0 and Turbo Vision library. It was my first serious programming experience. After creating AObmen system even my father believed that I can be a programmer :)
